SHIPMENTS by AIR.
LENGTH x BREATH x HEIGHT x NUMBER OF BOXES the whole divided by 6000 if the cargo dimension is in CENTIMETERS = WEIGHT OF THE CARGO IN KGS.
( * * * ) / 6000 in KGS
LENGTH x BREATH x HEIGHT x NUMBER OF BOXES the whole divided by 366 if the cargo dimension is in INCHES = WEIGHT OF THE CARGO IN KGS.
( * * * ) / 366 in KGS
SHIPMENTS by SEA.
LENGTH/100 x BREATH/100 x HEIGHT/100 x NUMBER OF BOXES = VOLUME OR THE CARGO IN CUBIC METERS.
/ 100 * / 100 * / 100 * in CUBIC METERS
